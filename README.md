 Enia Magist Project

 Project Overview
This project analyzes the Enia Magist dataset to extract meaningful insights, identify patterns, and support data-driven decisions.

 Objectives
- Perform exploratory data analysis (EDA) on the dataset
- Identify trends and patterns in the data
- Visualize key metrics for better understanding
- Provide actionable insights from the analysis

 Tools & Skills
- Python
- pandas, numpy
- matplotlib, seaborn
- Data visualization and EDA techniques

 Key Insights
- Specific patterns and trends identified in the dataset
- Visualizations highlight important relationships between features
- Findings can guide decision-making and strategy
